# How to use

- docker compose up -d
- mvn install
- cd target
- java -jar cacheproxy-0.0.1-SNAPSHOT.jar -p HTTP_ENDPOINT
- java -jar cacheproxy-0.0.1-SNAPSHOT.jar --clean-cache (To clean redis)
  <br/>

## Objective

<p> Serve as a proxy to retrieve cached data before making a new request</p>
